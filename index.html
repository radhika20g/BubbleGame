How to play ?

1. Identify and select numbers from bubbles on the screen.

2. Match the selected number with the one displayed in "Hit" in the top bar for a 10-point score increase.

3. Each successful match updates the top bar number and changes all bubble numbers.

4. Complete the game within a 60-second time limit, aiming for maximum matches to score higher.
